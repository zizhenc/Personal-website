  6. VoiceXML Studying and Testing
    7. Time: May, 2012    Lecturer: Frank P. Coyle
            I learnt VoiceXML from CSE 7347/5347: XML and Enterprise of Frank P. Coyle. VoiceXML is the W3C's standard XML format for specifying interactive voice dialogues between a human and a computer. It allows voice applications to be developed and deployed in an analogous way to HTML for visual applications. And if you go to Voxeo's Global Developer Portal website, you can create an account and get a phone number which can be to do the interactive voice dialogue work. That means you can write your own VoiceXML file and call the phone number to hear the dialogue you designed in your own VoiceXML file. I made a test of it which reports some News I created. You can call the number: (281)-407-5732 to hear the content.
