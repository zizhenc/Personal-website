  2. Matula Number

    Resources: Matula Number Generator

    Description: This is defined by the following algorithm. Here `pi(x)` is the number of primes less than or equal to `x`. 

    	matula(`n`):
		create a node(labelled `n`)
		for each prime factor `p` of `n`:
			add the subtree matula(`pi(p)`), by an edge labelled `p`
		return the node

  3. Primordial Tree

    Resources: Primordial Tree Generator

    Description: Take any pair of natural numbers, say `i` and `j`. Construct the two rooted trees corresponding to numbers `i` and `j`. Join them by a new edge labeled `p_i` n one direction and `p_j` in the other such that we form the two related products (`i times p_j`) and (`j times p_i`) associated with the two nodes of the new edge. That tree is then the tree generated by that original number pair `i` and `j`. This describes a new relation (`ip_j RR jp_i`) as a cluster of natural numbers which is an expoloration of original Matula Number of rooted tree.
