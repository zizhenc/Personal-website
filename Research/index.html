  2. Smallest-Last Coloring Algorithm on Random Geometric Graph to determine Backbones in Wireless Sensor Networks
    3. Time: October ~ December, 2012    Professor: David W. Matula    Project Report: Download
            Like any sentient organism, the smart environment relies first and foremost on sensory data from the real world. These sensory data could be provided by Wireless Sensor Networks (WSN’s). The nodes communicate wirelessly in WSN can be modeled as vertices in a Random Geometric Graph (RGG). This project displays a demonstration of selecting WSN backbones based Smallest-Last Coloring (SLC) algorithm process in a RGG model. The SLC algorithm contains two procedure: (1) Smallest-Last Ordering (SLO); (2) Greedy Coloring. SLO is an ordering alogrithm created by Professor David W. Matula in 1983 (Paper). I also created an animation demo of SLO alogrithm (YouTube). After SLO procedure, by applying a greedy coloring alogrithm according to the ordered nodes, we will get several independent sets (each color set is an independent set). Combining a pair of independent sets will give us a backbone (may be not perfect) of the WSN. I uploaded a running result demo of determining backbones in WSN on a Sphere topology (which could represent a global WSN) on YouTube (Link).
